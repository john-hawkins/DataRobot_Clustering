Prediction Explanation Clustering

Use this application to run clustering over the prediction explanations generated by any model produced by a DataRobot project.

Once generated, the results will be saved in the folder /static and reused if you request them again. The file name will be as follows: 

   PRJECT_ID-MODEL_ID--.png

If you want to regenerate te results for the same model with an alternative dataset you will need to delete the old version.
